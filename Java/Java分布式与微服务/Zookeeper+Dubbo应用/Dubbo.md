## HTTP和RPC对比

Remote Procedure Call （RPC） 远程过程调用

* RPC通常指框架，可以使用HTTP通信协议
* RPC传输效率快
* HTTP通用性高
* RPC一般自带负载均衡

如果项目都是Java，希望性能比较好，用户对延迟比较敏感，就用PRC框架比如Duboo
